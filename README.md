## Requirements
* Node.js

## Installation ##
* Node.js modules:<br>
  `npm i`
* Create a .env file in the base folder and paste it there:
 
	  token=YOUR_DISCORD_TOKEN
	  clientid=YOUR_APP_ID
  
* Run bot:<br>
  `npm run start`
